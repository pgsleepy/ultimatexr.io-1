# IUxrGrabbable.Placed Event
 

Event called right after the object was placed. An object is placed when the last grip is released and there is a compatible <a href="T_UltimateXR_Manipulation_UxrGrabbableObjectAnchor">UxrGrabbableObjectAnchor</a> near enough to place it on. The grab event parameters use the same values as Placed.
