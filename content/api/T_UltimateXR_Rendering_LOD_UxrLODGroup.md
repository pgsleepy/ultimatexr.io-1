# UxrLODGroup Class
 


Component that, added to a GameObject with a Unity LODGroup component, will take over the LOD switching.
 When using a locomotion based on teleportation, it will only switch LOD levels when the teleportation happens to avoid popping due to head movement. When using a smooth locomotion system it will use regular LOD switching.
